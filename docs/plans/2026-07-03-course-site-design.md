# MSDS 682 Course Site Design

Goal: create a private, editable course website scaffold for MSDS 682 Summer 2026.

The site should follow the simple academic style of the CMU AI Agents course page: a course title, term line, compact navigation, and readable content pages. It should avoid framework overhead while the course is still being designed.

Architecture:

- `index.html` provides the shell, navigation, and content mount point.
- `script.js` stores page content as structured data and renders routes using hash navigation.
- `styles.css` defines a restrained academic visual system.

Initial pages:

- Home: overview, logistics, themes, learning goals.
- Schedule: week-by-week placeholder schedule.
- Assignments: draft assignment list.
- Syllabus: policies and expectations placeholders.
- Staff: instructor and support placeholders.

Publishing is intentionally left disabled while the repository is private and the course design is in progress.
