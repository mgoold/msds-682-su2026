# MSDS 682 Summer 2026

Private working repository for the MSDS 682 Summer 2026 course website.

The site is a lightweight static course page inspired by simple academic course sites:

- Home
- Schedule
- Assignments
- Syllabus
- Staff

## Local Preview

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Publishing

GitHub Pages is not enabled yet. Keep the repository private while the course is being designed.
